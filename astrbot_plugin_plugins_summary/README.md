# 📋 插件功能汇总

![Version](https://img.shields.io/badge/Version-1.0.0-blue?style=flat-square)

> 🌟 汇总所有已安装插件的功能和命令，方便用户快速了解和使用所有插件

---

## 🍥 核心功能
- 📋 显示所有已安装插件的列表
- 🔍 查看指定插件的详细信息
- ⚙️ 汇总所有插件的命令
- 🔄 刷新插件列表信息

---

## ⌨️ 使用命令

### 1. 查看插件列表
```plaintext
/插件列表
```
显示所有已安装插件的基本信息，包括名称、描述和命令数量。

### 2. 查看插件详情
```plaintext
/插件详情 插件名称或序号
```
显示指定插件的详细信息，包括：
- 插件ID
- 名称和描述
- 版本和作者
- 所有命令和用法

### 3. 查看所有命令
```plaintext
/所有命令
```
汇总显示所有插件的命令，按插件分类，包含命令名称、描述和用法。

### 4. 刷新插件列表
```plaintext
/刷新插件列表
```
重新加载所有插件的信息，获取最新状态。
